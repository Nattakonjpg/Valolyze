Abstract



  ในปัจจุบันเกมส์ออนไลน์กำลังเป็นที่นิยมอย่างต่อเนื่อง โดยเป็นผลมาจากการเข้าถึงง่ายของอุปกรณ์คอมพิวเตอร์และการเชื่อมต่ออินเทอร์เน็ตที่สะดวก มีประเภทเกมส์ออนไลน์หลากหลายรูปแบบ เช่น เกมส์ยิงปืนมัลติเพลเยอร์ (Multiplayer First-Person Shooter) เช่น Counter-Strike: Global Offensive, Valorant การแข่งขันกันกับผู้เล่นทั่วโลกในการแข่งขันระดับมืออาชีพจะเรียกว่า วงการ E-sports หรือ Electronic Sports ที่ผู้เล่นมีความสามารถและความเชี่ยวชาญในการเล่นเกมเพื่อการแข่งขันในระดับสูง การเติบโตของวงการ E-sports เติบโตอย่างก้าวกระโดดในวงการกีฬาและสังคมทั่วโลก ส่งผลให้การแข่งขันในระดับมืออาชีพ ได้รับการ สนับสนุนและลงทุนจากบริษัทใหญ่ และมีความสนใจจากผู้เล่นและผู้ชมทั่วโลก โดยสิ่งที่ทำให้ทีมของผู้เล่นมีข้อ ได้เปรียบในการแข่งขันคือข้อมูลเกี่ยวกับการเล่นเกมส์ของผู้เล่น ที่เป็นข้อมูลที่ได้ผ่านมาจากการวิเคราะห์จากผู้เล่น และผู้เชี่ยวชาญ จะเป็นตัวแปรสำคัญที่ช่วยฝึกซ้อมทีมมีโอกาสชนะมากกว่าทีมคู่แข่ง
  ในปัจจุบันเกมส์ส่วนใหญ่ที่วิเคราะห์เช่น CSGO หรือ DOTA2 จะมี Replay file และ log fileที่จะบอกข้อมูลเกมส์ตั้งแต่ต้นจนจบให้ดาวร์โหลดอย่างเป็นสาธารณะแต่ในเกมส์ที่ผู้วิจัยจะนำมาวิเคราะห์คือเกมส์ Valorant ที่ไม่มี Replay file หรือ log fileที่เป็นสาธารณะให้ดาวร์โหลดโดยการวิเคราะห์ต้องวิเคราะห์ จากวิดิโอบันทึกการเล่นทำให้ใช้เวลานานหรือวิเคราะห์ได้ไม่เต็มประสิทธิภาพ
  ผู้วิจัยจึงเกิดความสนใจที่จะสร้างระบบวิเคราะห์เกมส์ Valorantที่จะสามารถแยกการกระทำ(Action) ออกเป็น ซื้ออุปกรณ์(Buy) ยิง(Shoot) การวางระเบิด(Plant) ใช้สกิล(Use Skill) กู้ระเบิด(Defuse) เก็บลูกแก้ว(Collect Orb) ตาย(Death) จากวิดิโอบันทึกการเล่นเนื่องจากเหตุการณ์ที่น่าสนใจมักจะเกิดการกระทำ(Action) เพื่อที่จะช่วยให้ผู้ใช้สามารถวิเคราะห์การกระทำของผู้เล่นและค้นหาเหตุการณ์ที่ส่งผลต่อการกระทำของผู้เล่นได้อย่างมีประสิทธิภาพ
  ในส่วนของงานวิจัยนี้ผู้วิจัยจึงพัฒนาตัวแบบโมเดล Machine Learning สำหรับการแยกการกระทำของผู้เล่น (Action) จากวิดิโอบันทึกการเล่นเกมส์ Valorant โหมด Competitive เพื่อลดระยะเวลาและเพิ่มประสิทธิภาพในการวิเคราะห์
